ZOHO SalesIQ ChatBot (ReactJS + Gemini API)

This project is a custom AI-powered chatbot built for ZOHO SalesIQ, using the Gemini 2.5 Flash Lite model from Google AI Studio.
It provides intelligent responses, smooth UI interactions, and full responsiveness across mobile, tablet, and desktop devices.

Project Overview

This chatbot integrates Google Gemini API with a ReactJS frontend and works seamlessly inside ZOHO SalesIQ.

Because the project uses the Google Gemini Free Tier API Key, API usage must be handled carefully.
The free-tier limits are:

15 requests/min

250K tokens/min

1,000 requests/day

⚠️ Please use it responsibly to avoid hitting quota limits.
